# Cypress e2e test

This repo contains code to run a e2e testing.

Dependencies:
´´´
python3-pip
xvfb
libgbm-dev
npm install --save-dev cypress-image-snapshot --legacy-peer-deps
´´´

To run the e2e test:
´´´
npm run cy:open
